# my_resume
This is my resume/CV, deployed on Digitial Ocean's App Platform.

#### Why take this approach?
It's easy and familiar to me. (Also, friends don't let friends develop websites using PHP or WordPress. It's 2024.)

# Techstack
This site is deployed with the following frameworks/packages.
- Python 3.12 or higher
- Wagtail 6.2 or higher
- Wagtail Resume (for the human-readable and friendly Resume/CV format)
- Wagtail Code Block (for showing of snippets using PrismJS)
